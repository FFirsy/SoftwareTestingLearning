# 一、什么是 Token

Token 是：

```
登录成功后服务器返回的身份凭证
```

例如：

```
token = abc123xyz
```

之后所有接口都需要：

```
Authorization: token
```

---

# 二、典型接口流程

真实系统流程：

```
登录接口
↓
获取 token
↓
访问其他接口
```

---

# 三、登录接口示例

请求：

```
POST /login
```

Body：

```
{
 "username":"admin",
 "password":"123456"
}
```

返回：

```
{
 "code":200,
 "msg":"login success",
 "token":"abc123"
}
```

---

# 四、Python 登录获取 token

示例代码：

```python
import requests

url = "https://example.com/login"

data = {
    "username": "admin",
    "password": "123456"
}

response = requests.post(url, json=data)

result = response.json()

token = result["token"]

print(token)
```

# 五、带 token 请求接口

请求头：

```
Authorization: token
```

Python 代码：

```python
headers = {
    "Authorization": token
}

url = "https://example.com/user/info"

response = requests.get(url, headers=headers)

print(response.json())
```

# 六、完整自动化脚本示例

```python
import requests

# 登录接口
login_url = "https://example.com/login"

login_data = {
    "username": "admin",
    "password": "123456"
}

login_response = requests.post(login_url, json=login_data)

login_result = login_response.json()

token = login_result["token"]

# 用户信息接口
info_url = "https://example.com/user/info"

headers = {
    "Authorization": token
}

info_response = requests.get(info_url, headers=headers)

assert info_response.status_code == 200

print("接口测试成功")
```

# 八、今日练习题

写一个 Python 程序：

请求：

```
https://jsonplaceholder.typicode.com/posts
```

要求：

1 发送 GET 请求  
2 断言状态码 200  
3 打印第一条数据的 title

---

答：

```python
import requests
url = "https://jsonplaceholder.typicode.com/posts"

response = requests.get(url)

data = response.json()

assert response.status_code == 200, "状态码错误"
assert len(data) > 0, "返回数据为空"
print(data[0]["title"])

```
