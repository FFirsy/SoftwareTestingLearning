# Day18：pytest 自动化测试框架

---

# 一、什么是 pytest

pytest 是：

Python 最流行的测试框架

作用：

```
用来管理和运行自动化测试用例
```

---

# 二、为什么要用 pytest

之前你写的是：

```python
assert ...
```

问题：

```
不能批量运行
没有测试报告
不好管理
```

pytest 可以：

```
自动运行所有测试
生成测试报告
管理测试用例
```

---

# 三、安装 pytest

```
pip install pytest

```
---

# 四、第一个 pytest 测试

创建文件：

```
test_demo.py
```

代码：

```python
def test_add():
    assert 1 + 1 == 2
```

运行：

```
pytest
```

---

# 五、pytest 命名规则（非常重要）

必须遵守：

```
文件名：test_*.py
函数名：test_*
```

否则不会执行！

# 六、接口自动化 + pytest

示例：

```python
import requests

def test_get_users():
    url = "https://jsonplaceholder.typicode.com/users"

    response = requests.get(url)

    assert response.status_code == 200
```

---

# 七、多个测试用例

```python
import requests

def test_users():
    res = requests.get("https://jsonplaceholder.typicode.com/users")
    assert res.status_code == 200

def test_posts():
    res = requests.get("https://jsonplaceholder.typicode.com/posts")
    assert res.status_code == 200
```

# 九、今日练习题

使用 pytest 写一个接口测试：

请求：

```
https://jsonplaceholder.typicode.com/posts/1
```

要求：

1 使用 pytest 写函数  
2 断言状态码 200  
3 断言 userId == 1  

---

答：

```python
import requests
def test_status_code():
    
    url = "https://jsonplaceholder.typicode.com/posts/1"

    response = requests.get(url)

    data = response.json()
    assert response.status_code == 200 , "状态码错误"
    assert data["userId"] == 1 ,"userId错误"
```
