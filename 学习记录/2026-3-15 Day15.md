# Day15：Python接口自动化（requests库）

---

# 一、什么是 requests

requests 是：

```
Python 最常用的 HTTP 请求库
```

作用：

```
用 Python 发送接口请求
```

例如：

```
GET
POST
PUT
DELETE
```

---

# 二、安装 requests

如果没有安装：

```
pip install requests
```

---

# 三、发送 GET 请求

示例代码：

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

print(response.text)
```

运行后会返回：

```
JSON数据
```

---

# 四、获取状态码

```python
print(response.status_code)
```

例如：

```
200
```

---

# 五、解析 JSON

接口返回 JSON：

```python
data = response.json()

print(data)
```

获取字段：

```python
print(data[0]["name"])
```

---

# 六、发送 POST 请求

示例：

```python
import requests

url = "https://jsonplaceholder.typicode.com/posts"

data = {
    "title": "test",
    "body": "hello",
    "userId": 1
}

response = requests.post(url, json=data)

print(response.json())
```

---

# 七、接口测试示例

```python
import requests

url = "https://jsonplaceholder.typicode.com/posts"

data = {
    "title": "test"
}

response = requests.post(url, json=data)

assert response.status_code == 201
```

这就是：

```
自动化测试断言
```

# 八、今日练习题

写一个 Python 程序：

发送 GET 请求：

```
https://jsonplaceholder.typicode.com/users
```

要求：

1 打印状态码  
2 打印返回 JSON 数据  

---

答：

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

# 打印状态码
print("状态码:", response.status_code)

# 解析JSON
data = response.json()

# 打印JSON数据
print("返回数据:", data)

```
