# Day16：Python接口自动化断言

---

# 一、什么是断言（Assertion）

断言就是：

验证程序运行结果是否符合预期。

如果不符合：

程序会直接报错。

---

# 二、Python 断言语法

```python
assert 条件
```

示例：

```python
assert 1 + 1 == 2
```

如果条件为 False：

程序会报错：

```
AssertionError
```

---

# 三、接口自动化中的断言

接口请求示例：

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

assert response.status_code == 200
```

如果返回状态码不是 200：

程序就会报错。

---

# 四、断言返回 JSON 数据

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

data = response.json()

assert response.status_code == 200
assert data[0]["name"] == "Leanne Graham"
```

---

# 五、完整接口测试示例

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

# 解析 JSON
data = response.json()

# 断言状态码
assert response.status_code == 200

# 断言返回数据
assert data[0]["name"] == "Leanne Graham"

print("接口测试通过")
```

运行成功输出：

```
接口测试通过
```

---

# 六、真实测试工程师写法（推荐）

```python
import requests

url = "https://jsonplaceholder.typicode.com/users"

response = requests.get(url)

data = response.json()

# 状态码断言
assert response.status_code == 200, "状态码错误"

# 数据断言
assert "name" in data[0], "返回数据缺少 name 字段"

print("测试通过")
```

---

# 七、为什么自动化测试必须使用断言

自动化测试必须：

自动判断是否成功。

否则只是：

打印日志，而不是测试。

# 八、今日练习题

写一个 Python 接口测试脚本：

请求：

```
https://jsonplaceholder.typicode.com/posts/1
```

要求：

1 发送 GET 请求  
2 断言状态码是 200  
3 断言返回 JSON 中 userId 等于 1  

---

答：

```python
import requests

url = "https://jsonplaceholder.typicode.com/posts/1"

response = requests.get(url)

data = response.json()

assert response.status_code == 200, "状态码错误"
assert data["userId"] == 1, "userId 不正确"

print("接口测试通过")

```
